# Shoppers-Store 
(Work In Progress)
### Technology
----------
- Web API Core - 3.1, EF Core [Code First - EF Transaction]

- Repository pattern

- JWT (Token based Authentication/Authorization)

- Sql Server

- React 18, Html, CSS, Javascript, Bootstrap 5, Google chart api
